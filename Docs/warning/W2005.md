# W2005 - 回退到备用方法

## 代码信息

- **代码**: `W2005`
- **类型**: 警告
- **等级**: WARNING
- **描述**: 首选方法失败，已回退到备用实现

## 含义

此警告表示引擎尝试使用首选方法执行操作失败，已自动切换到备用方法。功能仍可用，但性能或用户体验可能下降。

## 触发条件

- Gum 库不可用，回退到控制台输入
- Gum 选择菜单抛出异常，回退到数字输入
- 首选的文件访问方法失败
- 调试模式未启用时尝试使用调试功能

## 示例输出

```
[2024-01-01 12:00:00] WARNING [W2005] Gum not available, falling back to original method
[2024-01-01 12:00:00] WARNING [W2005] Falling back to original menu display
[2024-01-01 12:00:00] WARNING [W2005] The Debug is not enabled
[2024-01-01 12:00:00] WARNING [W2005] Falling back to original selection method
```

## 控制台输出（回退到数字输入）

```
请选择：
1. 继续游戏
2. 保存并退出
3. 不保存退出
请输入数字选择 (1-3): 
```

## 解决方案

1. **对于 Gum 回退**：安装 Gum 库以获得更好的交互体验
   
   ```
   winget install charmbracelet.gum
   ```

2. **对于调试模式**：在 `data.cfg` 中启用调试模式
   
   ```ini
   DevModeEnabled = 1
   ```
