# W2003 - 版本不匹配

## 代码信息

- **代码**: `W2003`
- **类型**: 警告
- **等级**: WARNING
- **描述**: 已安装的插件版本与脚本要求的版本不兼容

## 含义

此警告表示脚本中通过 `use` 命令指定的插件版本与已安装的插件版本不一致。游戏仍会尝试执行，但可能存在兼容性问题。

## 触发条件

- 脚本中使用 `use plugin_name version` 指定版本
- 插件的 `about.cfg` 中包含 `Version` 字段
- 要求的版本与已安装的版本不匹配

## 版本比较规则

- 支持语义化版本号（如 `1.0.0`、`2.1.3`）
- 主版本号不同时发出警告
- 次版本号高于要求时允许执行
- 非数字版本号使用精确匹配

## 示例输出

```
[2024-01-01 12:00:00] WARNING [W2003] Plugin version mismatch: required=audio_player v2.0.0, installed=v1.5.0
```

## 控制台输出

```
==============================
插件版本警告
脚本需要: audio_player v2.0.0
已安装: audio_player v1.5.0

警告：插件版本可能不兼容
建议安装 audio_player v2.0.0
==============================

按任意键继续（或按ESC返回主菜单）...
```

## 解决方案

1. 更新插件到脚本要求的版本
2. 修改脚本中的版本要求以匹配已安装的版本
3. 如果版本差异不影响功能，可以忽略警告
