# W2001 - 文件不存在

## 代码信息

- **代码**: `W2001`
- **类型**: 警告
- **等级**: WARNING
- **描述**: 尝试访问的文件不存在

## 含义

此警告表示引擎尝试打开、读取或操作一个不存在的文件。操作可能会继续执行，但功能可能受限。

## 触发条件

- 配置文件 `data.cfg` 缺失
- 游戏脚本文件 `.pgn` 不存在
- 存档文件不存在
- 结局文件 `data.inf` 不存在
- `show` 命令指定的文件不存在
- `safeViewFile()` 函数无法找到目标文件

## 示例输出

```
[2024-01-01 12:00:00] WARNING [W2001] File not found: Novel/NotFound/NotFound.pgn
[2024-01-01 12:00:00] WARNING [W2001] Tutorial file not found
[2024-01-01 12:00:00] WARNING [W2001] Key not found in configuration file: AutoRun
```

## 控制台输出

```
ERROR [W2001] FileError: File does not exist
       > Line 0: 
       Hint: Check if the file path is correct and the file exists
       See: https://github.com/Colasensei/PaperVisualNovel/tree/master/Docs/errors/W2001.md
```

## 解决方案

1. 检查文件路径是否正确
2. 确认文件是否存在于指定位置
3. 检查文件权限设置
4. 对于配置文件，可以创建默认的 `data.cfg`
