# E3002 - 未知命令

## 代码信息

- **代码**: `E3002`
- **类型**: 错误
- **等级**: ERROR
- **描述**: 脚本中包含引擎无法识别的命令

## 含义

此错误表示 PGN 脚本中使用了 PaperVisualNovel 引擎不支持的命令。这可能是由于拼写错误、使用了已弃用的命令或自定义命令未正确注册导致的。

## 触发条件

- 脚本行首的命令词不在引擎支持的命令列表中
- 命令名称拼写错误
- 使用了其他引擎的专有命令

## 支持的命令列表

| 命令                   | 功能     |
| -------------------- | ------ |
| `say`                | 显示对话文本 |
| `choose`             | 显示选择菜单 |
| `set`                | 设置变量   |
| `random`             | 生成随机数  |
| `if`                 | 条件判断   |
| `jump`               | 跳转到指定行 |
| `end`                | 结束游戏   |
| `endname`            | 记录结局   |
| `wait`               | 等待指定时间 |
| `cls`/`clean`        | 清屏     |
| `show`               | 显示文件   |
| `input`              | 获取用户输入 |
| `plugin`/`runplugin` | 执行插件   |
| `use`                | 声明插件依赖 |

## 示例输出

```
[2024-01-01 12:00:00] ERROR [E3002] Unknown command: pritn
[2024-01-01 12:00:00] ERROR [E3002] Unknown command: gotoo
[2024-01-01 12:00:00] ERROR [E3002] Invalid menu selection, default to continue
```

## 控制台输出

```
ERROR [E3002] CommandError: Unknown PGN command
       > Line 123: pritn "Hello World" 0.5 white
         ^~~~~~
       Hint: Check the command spelling or refer to the documentation for valid commands
       See: https://github.com/Colasensei/PaperVisualNovel/tree/master/Docs/errors/E3002.md
```

## 解决方案

1. 检查命令拼写是否正确
2. 参考支持的命令列表使用正确的命令
3. 如果是自定义命令，需要注册到引擎
4. 注意命令大小写（不区分大小写）
