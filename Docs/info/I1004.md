# I1004 - 插件加载成功

## 代码信息

- **代码**: `I1004`
- **类型**: 信息
- **等级**: INFO
- **描述**: 插件成功加载

## 含义

此信息表示插件管理器成功读取并验证了插件的配置文件，插件已准备就绪可供使用。

## 触发条件

- 启动时扫描 `Plugins/` 目录
- 成功读取插件的 `about.cfg` 配置文件
- 插件包含必要的 `RunCommand` 和 `RunFile` 字段

## 示例输出

```
[2024-01-01 12:00:00] INFO [I1004] Reading installed plugins from: Plugins
[2024-01-01 12:00:00] INFO [I1004] Plugin loaded: image_viewer v1.0.0
[2024-01-01 12:00:00] INFO [I1004] Plugin dependency registered: audio_player v2.1.0
[2024-01-01 12:00:00] INFO [I1004] Plugin executed successfully: calculator (took 45ms)
```

## 插件配置格式

```ini
# about.cfg
RunCommand = python
RunFile = main.py
Description = "图像查看器插件"
Version = 1.0.0
Author = colaSensei
```
