# I1005 - 结局保存成功

## 代码信息

- **代码**: `I1005`
- **类型**: 信息
- **等级**: INFO
- **描述**: 结局成功记录到收集系统

## 含义

此信息表示玩家达成了一个新结局，该结局已被成功记录到游戏的结局收集系统中。

## 触发条件

- 脚本中执行 `endname` 命令
- 结局名称不为空
- 该结局尚未被收集过

## 示例输出

```
[2024-01-01 12:00:00] INFO [I1005] Ending saved: 真相大白
[2024-01-01 12:00:00] INFO [I1005] Collected 3/10 endings for game: HelloWorld
[2024-01-01 12:00:00] INFO [I1005] Registered 10 total endings from script
```

## 结局存储

结局信息默认保存在游戏目录的 `data.inf` 文件的 `[ENDINGS]` 部分：

```ini
[ENDINGS]
真相大白
永远的朋友
新的开始
```
