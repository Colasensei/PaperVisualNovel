# I1001 - 游戏启动

## 代码信息

- **代码**: `I1001`
- **类型**: 信息
- **等级**: INFO
- **描述**: 游戏引擎成功启动

## 含义

此信息表示 PaperVisualNovel 引擎已成功初始化并开始运行。这是引擎启动时的正常日志信息。

## 触发条件

- 主程序 `main()` 函数开始执行
- 控制台标题设置完成
- 随机数种子初始化完成
- Gum 库检查通过

## 示例输出

```
[2024-01-01 12:00:00] INFO [I1001] The program is running...
[2024-01-01 12:00:00] INFO [I1001] No command line arguments, starting normal flow
```
