# I1003 - 游戏保存成功

## 代码信息

- **代码**: `I1003`
- **类型**: 信息
- **等级**: INFO
- **描述**: 游戏状态成功保存到存档文件

## 含义

此信息表示游戏引擎成功将当前游戏状态（变量、选择历史、结局收集等）保存到存档文件中。

## 触发条件

- ESC 菜单中选择"保存并退出"
- 自动存档机制触发
- `saveGame()` 函数成功执行

## 示例输出

```
[2024-01-01 12:00:00] INFO [I1003] Start to save game
[2024-01-01 12:00:00] INFO [I1003] Game saved successfully: Novel/HelloWorld/saves/autosave.sav (1024 bytes, took 15ms)
[2024-01-01 12:00:00] INFO [I1003] Save file deleted
```

## 存档位置

存档文件默认保存在游戏脚本所在目录的 `saves` 文件夹中，默认文件名为 `autosave.sav`。
