# I1002 - 游戏加载成功

## 代码信息

- **代码**: `I1002`
- **类型**: 信息
- **等级**: INFO
- **描述**: 游戏脚本或存档加载成功

## 含义

此信息表示游戏引擎成功加载了 PGN 脚本文件或从存档恢复了游戏状态。

## 触发条件

- `RunPgn()` 函数成功打开并读取 PGN 文件
- `loadGame()` 函数成功读取并解析存档文件
- 游戏从主菜单选择后成功加载

## 示例输出

```
[2024-01-01 12:00:00] INFO [I1002] Preparing to run game HelloWorld.pgn
[2024-01-01 12:00:00] INFO [I1002] Save file loaded
[2024-01-01 12:00:00] INFO [I1002] Game choose: HelloWorld
```

## 性能信息

此信息通常会附带性能评估日志 [D4001](D4001.md)，显示文件读取时间：

```
DEBUG [D4001] Read 1683 lines from game file (took 23ms 32KB)
```
